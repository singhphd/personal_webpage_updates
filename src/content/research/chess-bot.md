## Project Goal

To explore the capabilities of deep reinforcement learning by building a chess engine from scratch. This project serves as a testbed for understanding neural network architectures and search algorithms.

## Implementation Details

- **Framework:** PyTorch
- **Architecture:** Convolutional Neural Network (CNN) for board evaluation.
- **Training:** Self-play reinforcement learning.
- **Search:** Monte Carlo Tree Search (MCTS) (Planned/In-progress).

## Learning Outcomes

This project honed my skills in:
- Designing custom loss functions.
- Managing training data pipelines.
- Optimizing model inference for real-time decision making.
